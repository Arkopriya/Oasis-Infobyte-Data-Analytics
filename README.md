# Retail Sales EDA

## Objective
Perform Exploratory Data Analysis (EDA) on a retail sales dataset to uncover customer behavior, sales trends, and business insights.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Features
- Data inspection and quality check
- Descriptive statistics
- Monthly and quarterly sales analysis
- Customer demographics analysis
- Product category analysis
- Correlation heatmap
- Business insights and recommendations

## Outcome
Identified sales patterns and customer trends to support data-driven business decisions.
